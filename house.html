House of Hair

The brief was to update my client's current hair salon website. He wanted something modern and stylish; something that would reflect his brand. I find it most useful to design in the browser so that the client can really get a feel for the site and how it would look when it goes live.

With this in mind, I used html and css to create a number of versions of the website and then used Notable to display my designs. The use of Github made it easy for me to keep all my code in case something needed changing.
